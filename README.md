# prenis
.obj viewer based on OpenGL
# How to use
Add obj\default.obj to main executable or use
``prenis.exe obj\whatever.obj``
# Controls
Arrows - move camera  
F1 - reset camera  
F2 - change light  
F3 - change diffuse  
F4 - auto camera  
F5 - blinking background  
F6 - blinking object light
# Using
OpenGL  
Tiny obj loader  
stb_image
